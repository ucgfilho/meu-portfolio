# Meu Portfólio

Bem-vindo ao repositório do meu portfólio pessoal. Este projeto apresenta minhas habilidades, certificações e projetos na área de desenvolvimento e QA.

## Informações do Projeto

Este é um projeto web moderno focado em performance e design responsivo.

## Como executar este projeto?

Para rodar o projeto localmente em sua máquina, você precisará ter o [Node.js](https://nodejs.org/) instalado.

Siga os passos abaixo:

```sh
# Passo 1: Clone o repositório
git clone <URL_DO_SEU_REPOSITORIO>

# Passo 2: Entre na pasta do projeto
cd meu-portfolio

# Passo 3: Instale as dependências
npm install

# Passo 4: Inicie o servidor de desenvolvimento
npm run dev
